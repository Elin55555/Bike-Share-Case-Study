# Bike Share Case Study

# Google Data Analytics Capstone: Cyclistic Bike-Share Analysis

## Project Overview
This case study analyzes historical trip data from Cyclistic, a fictional bike-share company, to identify differences between casual riders and annual members. The goal is to design marketing strategies to convert casual riders into annual members.

## Tools Used
- **Language:** R
- **IDE:** RStudio
- **Libraries:** tidyverse, lubridate, ggplot2
